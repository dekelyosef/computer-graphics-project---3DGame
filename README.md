# הסבר כללי
openGL יצירת משחק תלת מימדי תוך שימוש בממשק

המשחק מאפשר תנועה של השחקן במרחב בתוך מרחב תלת מימדי סגור המורכב מ-2 חללים, זיהוי התנגשויות וכולל שימוש בטקסטורה, תאורה ומספר אובייקטים נעים (מלשון תנועה)
